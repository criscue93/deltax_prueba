## Description

App to be able to start a project with NestJs, with connection to Mysql and MongoDB.

## Installation

```bash
$ npm install
```

## .ENV

In the .env.example file are the environment variables to be used to run the project.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
